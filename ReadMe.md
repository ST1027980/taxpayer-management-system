# Taxpayer Management System

## Overview
A responsive frontend simulation of a public-sector system used to manage taxpayer records. The project demonstrates data handling, validation, and client-side storage.

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## Features
- Taxpayer registration
- Input validation
- Persistent data using LocalStorage
- Delete functionality
- Responsive design
- Clean, government-style UI

## Purpose
This project was built to demonstrate frontend development skills aligned with public service and enterprise systems.
